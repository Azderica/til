- 중복 코드가 가지는 가장 큰 문제는 코드를 수정하는 데 필요한 노력을 몇 배로 증가시키는 것입니다.

- DRY는 **Don't Repeat Yourself, 반복하지 마라**의 의미를 가집니다.
- DRY 원칙은 **한 번, 단 한번(Once and Only Once) 원칙** 또는 **단일 지점 제어(Single-Point Control) 원칙**이라고 부릅니다.
