# 2. 데이터 모델과 질의 언어

> 내 언어의 한계는 내 세계의 한계를 의미합니다.

- 데이터 모델은 소프트웨어가 어떻게 작성됐는지 뿐만 아니라 **해결하려는 문제에 대해 고민**해봐야합니다.
- 각 계층의 핵심적인 문제는 다음 하위 계층 관점에서 데이터 모델을 **표현**하는 방법입니다.

각 계층은 명확한 데이터 모델을 제공해 하위 계층의 복잡성을 숨깁니다.
